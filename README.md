# ⚙️ EMS Backend API

The robust, RESTful Node.js API that powers the Employee Management System (EMS). It provides secure and efficient endpoints to manage employee records, handling all data operations seamlessly.

## ✨ Key Features

- **Full CRUD Architecture**: Supports complete Create, Read, Update, and Delete operations for employee records.
- **RESTful Endpoints**: Clean and structured API routes for easy integration with any frontend framework.
- **CORS Configured**: Fully configured Cross-Origin Resource Sharing to allow seamless communication with the frontend dashboard.
- **Lightweight & Fast**: Built for speed and reliability, ensuring real-time UI updates on the frontend are handled instantly.

## 🛠️ Tech Stack

- **Runtime**: Node.js
- **Framework**: Express.js
- **Data Handling**: JSON Data Structures

## 🚀 Running Locally

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
